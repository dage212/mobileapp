# mobileapp是一个网页版app,可以看视频，听歌，百度地图

技术栈 vue+museui+vue+router+laravel

实现功能 视频 音频 百度地图 


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
